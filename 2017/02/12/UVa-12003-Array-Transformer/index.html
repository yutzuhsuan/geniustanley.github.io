塊狀鍊表

12003 - Array Transformer

要對一個很大的 array 做操作的時候可以用塊狀鍊表

英文網路稱做 Unrolled Linked List ，中文網路稱作「鬆散鏈表」、「塊狀鏈表」。查無正式學術名稱。

複雜度可以去看 演算法筆記 - Data

舉例

以下以 sample input 舉例

  1. 每塊的 size 是 sqrt(10) = 3
  2. 開始分塊
  3. 對每塊 sort
  4. query 2 ~8
    • 2, 3      在 block1 // 有 1 參雜所以用 iterate
    • 4, 5, 6 在 block2 // binary search
    • 7, 8      在 block3 // 有 9 參雜所以用 iterate
  5. insert
    • 找到 10 在 block4 就去修改
    • 修改完要記得 sort

還有要順便記位置, 就不贅述了
