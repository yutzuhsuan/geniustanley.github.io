#UVa

雖然這題是 Recursive Backtracking，但是我覺得狀態壓縮的技巧比較像重點

416 - LED Test

解法

  1. 把 7 段顯示器的燈號換成用 binary 加總來存
  2. 把每個可能的 count down sequence 都嘗試過一次 ( 最多 10 次 )
  3. 把每次可能壞掉的燈號都嘗試過一次 ( 最多 2^7 = 127 種壞法 )
